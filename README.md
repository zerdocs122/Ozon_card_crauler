# Ozon Search Position Parser

Скрипт определяет позицию товара в поисковой выдаче Ozon по поисковому запросу и артикулу (SKU). Просматривает до 100 первых результатов (3 страницы).

## Зависимости

```
pip install -r requirements.txt
```

Для работы необходим установленный **Google Chrome**.

## Использование

```bash
python main.py "поисковый запрос" АРТИКУЛ [--show-browser] [--debug] [--output result.json]
```

| Флаг | Описание |
|---|---|
| `--show-browser` | Показать окно браузера (по умолчанию работает в фоне) |
| `--debug` | Сохранять скриншот каждой страницы |
| `--output FILE` | Сохранить результат в JSON-файл |

## Пример

```bash
python main.py "посудомоечная машина" 2384073618 --output result.json
```

**Пример вывода:**

```json
{
  "query": "посудомоечная машина",
  "sku": "2384073618",
  "position": 19,
  "page": 1,
  "total_checked": 100,
  "timestamp": "2026-06-06T19:52:30"
}
```

Если товар не найден в первых 100 результатах, `position` равен `"not_found"`.

## Тестовые данные

| Запрос | Артикул (SKU) | Почему выбран |
|---|---|---|
| `посудомоечная машина` | `2384073618` | Высокий средний чек, умеренная конкурентность — товар стабильно присутствует в топ-100 |
| `пылесос` | `4280736702` | Широкая категория с высокой конкуренцией — проверяет поиск среди большого числа похожих товаров |
| `футболка` | `1535661971` | Максимально широкая категория с тысячами товаров — проверяет сценарий `not_found` в первых 100 результатах |

## Как это работает

- Использует **nodriver** — управляет реальным Chrome без детектируемых следов автоматизации
- Обходит Cloudflare и антибот-защиту Ozon благодаря запуску настоящего браузера
- Прокручивает страницу для подгрузки lazy-loaded карточек перед сбором SKU
- SKU извлекается из URL карточки (`/product/название-товара-АРТИКУЛ/`) прямо в JavaScript
