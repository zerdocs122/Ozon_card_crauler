# Лог подхода

## Часть 1. Парсер

### Шаг 1 — Выбор стека

**Промпт AI:**
> Нужно написать Python-скрипт для парсинга поисковой выдачи Ozon (SPA на React).
> Требования: найти позицию товара по SKU в первых 100 результатах.
> Ozon использует Cloudflare и динамическую загрузку контента через JS.
> Сравни Playwright, Selenium и nodriver по критериям: обход антибота, зрелость API,
> асинхронность, простота настройки. Что выберешь и почему?

AI рекомендовал **Playwright** как современный инструмент с хорошим API и поддержкой
`playwright-stealth` для обхода антибота. `nodriver` упомянул как альтернативу
специально для Cloudflare, но охарактеризовал как «сложнее в настройке».

**Решение:** начать с Playwright, при блокировках — переходить на nodriver.

---

### Шаг 2 — Playwright: браузер не установлен

Первый запуск упал с ошибкой:

```
playwright._impl._errors.Error: BrowserType.launch: Executable doesn't exist at
C:\Users\...\chromium-1194\chrome-win\chrome.exe
```

Playwright устанавливает собственный Chromium отдельной командой:

```bash
playwright install chromium
```

Добавили `setup.bat` для Windows, чтобы одной командой ставить и зависимости, и браузер.

---

### Шаг 3 — Playwright: Cloudflare блокирует

После установки Ozon возвращал пустую страницу с кодом инцидента `fab_chlg_*`.

**Промпт AI:**
> Playwright с headless=True блокируется Cloudflare на Ozon — получаю код инцидента
> fab_chlg_* и пустую выдачу. Какие конкретно браузерные сигнатуры детектирует
> Cloudflare и как playwright-stealth их закрывает? Есть ли сигнатуры, которые
> playwright-stealth не покрывает?

**Попытка 1:** добавили `playwright-stealth`. Закрывает `navigator.webdriver`, WebGL vendor,
plugins и др. Результат: та же блокировка — Cloudflare детектировал по поведению,
не только по сигнатурам.

**Промпт AI:**
> playwright-stealth не помог. Cloudflare всё равно блокирует. Ozon использует
> поведенческий анализ. Какой следующий шаг: резидентный прокси, смена браузерного
> движка или переход на nodriver? Обоснуй выбор с учётом того, что нужен
> долгосрочный надёжный инструмент, а не разовый обход.

**Попытка 2:** купили резидентный прокси IPv4 Shared (proxy6.net, Россия).
Shared-прокси уже был в базах блокировок Cloudflare — страница загружалась, но выдача пустая.

**Попытка 3:** купили обычный IPv4. Прокси подключился, страница открылась,
но товары не появились — Cloudflare детектировал Playwright по поведению браузера
независимо от IP.

### Шаг 4 — Переход на nodriver

**Промпт AI:**
> Playwright не проходит Cloudflare даже с чистым резидентным IPv4 и playwright-stealth.
> Хочу перейти на nodriver. Напиши асинхронный скрипт на nodriver, который:
> 1. Открывает поисковую выдачу Ozon
> 2. Прокручивает страницу для загрузки lazy-loaded карточек
> 3. Извлекает SKU из href ссылок через JavaScript в браузере
> 4. Перебирает до 3 страниц (100 товаров)
> Добавь случайные задержки между действиями.

nodriver управляет **реальным Chrome**, установленным на компьютере — Cloudflare
не отличает скрипт от живого пользователя. Прокси не потребовался.

**Что изменилось при переходе:**
- Playwright (синхронный API) → nodriver (асинхронный, `async/await`)
- Скачиваемый Chromium → реальный Chrome пользователя
- Добавили случайные задержки и прокрутку для имитации поведения человека
- `--disable-blink-features=AutomationControlled` убирает флаг автоматизации

**Результат:** скрипт заработал, Ozon отдаёт выдачу.

## Часть 2. Устойчивость

Скрипт запускался 3 раза подряд с интервалом 30 секунд — все 3 запуска вернули результат.

**Что обеспечивает устойчивость:**

1. **nodriver + реальный Chrome** — Cloudflare не отличает скрипт от живого пользователя
2. **Случайные задержки** между страницами (1.5–3.5 сек) и шагами прокрутки (0.3–0.7 сек)
3. **Случайная прокрутка** (5 шагов по 300–500 px) — имитирует чтение страницы
4. **`--disable-blink-features=AutomationControlled`** — убирает флаг автоматизации

**Безопасные лимиты (оценка):**
- Не более 1 поискового запроса каждые 5–10 секунд
- Не более 200–300 запросов в день с одного IP
- При масштабировании — ротация IP через резидентные прокси
